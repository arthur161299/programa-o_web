# programa-o_web
PEOJETO VITE
O useState é o principal Hook do React para gerenciar o estado (dados mutáveis que afetam a renderização) em componentes funcionais.

tarefas: É a variável que armazena o estado atual, ou seja, o array com todos os itens da lista.

setTarefas: É a função que você usa exclusivamente para atualizar o valor de tarefas.

useState([]): Inicializa o estado com um array vazio, que é o valor inicial da lista.

Gerenciar o Campo de Input:

input: Armazena o texto atual digitado no campo de entrada.

setInput: Atualiza o valor do texto no campo de entrada.
